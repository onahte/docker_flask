Project Outline

The topics covered in class have allowed students to become familiar with the most in-demand web technologies, paving the way to an entry-level devops role. By implementing the tools learned here, the students can now automate pipelines and integrate unit testing to build robust workflows that reflect the needs of a real-world development environment. This website covers these tools and discusses how they have been applied to build this site (aka Project 1).
